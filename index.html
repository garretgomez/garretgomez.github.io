About Me

Hello my name is Garret Gomez. I am a free lance web developer and designer. I became fascinated with web design and development when I took an introduction course in college. After that course I wanted to learn more about web development and design so I took more courses. Some of the courses includes web design I, JavaScript, and Dreamweaver. Although theses would be the only courses I would take in college I continued to learn more by taking online courses and continuing to work on personal projects to help sharpen my skills. Additionally, I would take courses where I learned C# and SQL which helped learn new coding skills that transferred into helping me with web development. My main knowledge and expertise of web development is front-end although I do have some back-end experience such as SQL.
